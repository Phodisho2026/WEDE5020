# WEDE5020
Web development project
## Project Title
Phodii's flower pod

## Project Overview
Phodii's Flower Pod is a flower shop website based in South Africa. The website sells beautiful flower arrangements and bouquets with prices in ZAR. It includes an enquiry form, contact information, a shopping cart feature, and an image gallery with lightbox functionality.

## Website Goals
- Provide an attractive online presence for the flower shop
- Display products with clear pricing in South African Rand (ZAR)
- Allow customers to enquire about custom arrangements
- Share the shop's story, mission, and vision
- Show store locations with maps
- Display flower images in a gallery with lightbox

## Key Features
- Responsive design (works on desktop, tablet, and mobile)
- Shopping cart functionality
- Form validation for enquiries and contact
- Product gallery with prices
- Image gallery with lightbox
- Multiple store locations (Centurion only)
- SEO optimized with meta tags

## Pages

### Home (index.html)
- Welcome message and hero section
- Features section (Why Choose Us)
- Seasonal special promotion

### About Us (about.html)
- Our Story - background of the flower shop
- Mission and Vision statements
- What We Offer - services list
- Store Locations with Google Maps links

### Products (products.html)
- Product grid showing all bouquets
- Each product has name, description, price (in ZAR), and image
- Shopping cart functionality (add/remove items)
- Checkout feature

### Gallery (gallery.html)
- Image gallery displaying flower photos
- Lightbox feature - click image to view larger
- Responsive grid layout

### Enquiry (enquiries.html)
- Enquiry form with validation
- Fields: Full Name, Email, Subject, Message
- JavaScript validation before submission
- Success message after submission

### Contact (contact_us.html)
- Contact information (email, phone, hours)
- Two store locations with addresses
- Contact form with validation
